Фон карточки сделки

Цветовая метка для карточки CRM: палитра в поле сделки, вкладка «Цвет фона», автоподсветка при открытии. Удобно для брони, VIP, срочных и этапов воронки.

  • Поле-виджет «Цвет фона карточки» в карточке сделки
  • Вкладка с палитрой и сохранением в CRM
  • Роботы могут менять цвет при смене стадии
