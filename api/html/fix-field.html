Поле «Цвет фона карточки»

Переводит поле с типа string (текст #E3F2FD) на виджет с палитрой.

Кнопок нет — страница сама выполняет исправление, когда открыта из портала Битрикс24.

Вы открыли ссылку в обычной вкладке браузера. Так не работает — нужен контекст приложения на портале.

  1. Зайдите на b24-s2an91.bitrix24.ru
  2. Приложения → Разработчикам → Установленные
  3. Фон карточки сделки → Открыть
  4. На странице приложения — «Исправить поле (нет цвета)»

Или переустановите: install.html (тоже только из портала).

Если на сделке уже цветная заливка — эту страницу можно не открывать.
