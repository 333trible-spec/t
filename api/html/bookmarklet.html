Закладка — фон всей карточки

Если приложение не может покрасить карточку (ограничение браузера), один раз перетащите кнопку на панель закладок. На карточке сделки нажимайте закладку — фон подтянется из поля «Цвет фона карточки».

B24: подсветить фон

Как установить

  1. Покажите панель закладок (Ctrl+Shift+B в Chrome).
  2. Перетащите синюю кнопку на панель закладок.
  3. На портале откройте сделку → нажмите закладку.

После смены цвета в поле — снова нажмите закладку или F5.
